##Requirements:
- Node v6.11.0
- npm v3.10.10
- nodejs

## Server up:
Run (from directory where is package.json file)
```
npm install
```
Now run the test with:
```
node index.js
```

##To run test: go to test folder. And run 
```
npm test
```
